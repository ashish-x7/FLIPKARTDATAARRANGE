        How to use Create Folder

        This tool replicates the VBA folder grouping logic:

        1. Upload your **Flipkart Merged Orders** file (must contain Flipkart_Merged_Orders in the filename).
        2. Upload your renamed sheets starting with prefixes (e.g. 101-TaxReport..., 509-Invoice...).
        3. Click **Group & ZIP Folders**.
        4. The system creates folders for each prefix, copies the merged file inside each, moves the prefix file there, and packages everything in a sorted ZIP file named [firstPrefix]-[lastPrefix].zip!
        Note: Exactly one file must contain the word FLIPKART_MERGED_ORDERS in its name.

          Invoice Arrange Process

          Required Uploads (Exactly 3 files):

          • OD File: Contains FLIPKART_MERGED_ORDERS in the filename.
          • DT File: Contains TaxReportData in the filename.
          • Details File: Contains -FLIPKART and prefix (e.g. 101-Bharvita-FLIPKART...).

          Execution Steps:

          1. Matches Details file's Invoice No (Column B) in DT (Column G) and **deletes matched rows**.
          2. Calculates suffix range (e.g. 1-9) and saves remaining DT to a new file (e.g. 101-(2-9)-DT.xlsx).
          3. Highlights matching rows in OD (Column A) using DT's suborders (Column E) and exports them to a new file (e.g. 101-(FK27S101-2-9)-OD.xlsx).
          4. Performs VBA clean macro to export **GST NOT APPLICABLE.xlsx** (if Col G is not empty and Col AP is empty) and **2 MORE INVOICE.xlsx** (if duplicate invoices exist).
